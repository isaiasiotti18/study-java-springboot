# ☕ Java — Estudos

> Projetos Java são compilados e rodam independentemente (sem npm).

## Tópicos para revisar

- [ ] POO (herança, polimorfismo, interfaces, classes abstratas)
- [ ] Collections Framework
- [ ] Exception Handling
- [ ] Streams e Lambdas (Java 8+)
- [ ] Generics
- [ ] Spring Boot (API REST, JPA, Security)
- [ ] Testes (JUnit, Mockito)
- [ ] Design Patterns (Factory, Strategy, Observer)

## Como organizar

```
java/
├── 01-poo/
│   └── Main.java
├── 02-collections/
└── ...
```

## Como compilar e rodar

```bash
cd 01-poo
javac Main.java
java Main
```

Para projetos Spring Boot, use o [Spring Initializr](https://start.spring.io/) e coloque dentro da pasta correspondente.
